# 🔊 OpenVoice

> Менеджер приватных голосовых комнат для серверов Discord, написанный на JavaScript.

## ⚙️ Особенности проекта

* 🔒 **Приватность:** Автоматическое создание и конфигурация изолированных комнат.
* 📦 **Локальная БД:** Быстрая и легкая работа со структурированными данными через SQLite.
* 🎯 **Кастомная логика:** Бот оптимизирован для точечной работы на **2–3 серверах** (не рассчитан на глобальный публичный хостинг).

---

## 🛠️ Системные требования

Перед началом работы убедитесь, что у вас установлены:
* **Node.js** версии 16.x или выше
* Получен **Токен бота** и **ID Клиента** на портале [Discord Developer](https://discord.com/developers)

## 🚀 Быстрый запуск

### 1. Подготовка окружения (только для Windows)
Если при запуске скриптов PowerShell блокирует выполнение команд, разрешите запуск в текущей сессии:
```bash
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
```

### 2. Клонирование и установка
Склонируйте репозиторий на свой компьютер и установите все необходимые зависимости:
```bash
# Клонируем проект
git clone https://github.com/weeeklyx/openvoice.git

# Переходим в папку бота
cd openvoice

# Инициализируем окружение (если требуется)
npm init -y

# Устанавливаем библиотеки
npm install discord.js dotenv sqlite sqlite3
```

### 3. Настройка конфигурации
Создайте файл `.env` в корневой папке проекта и добавьте в него ваш секретный токен:
```env
TOKEN=ваша_строка_токена_дискорд_бота
```

### 4. Старт бота
Запустите главный файл скрипта:
```bash
node index.js
```
